# Kanji Readings Converter

Converts the JOYO Kanji Excel sheet into a JSON file of readings, frequencies, and example words.

## Requirements

- Python 3.12+
- Packages listed in requirements.txt

## Setup

```bash
pip install -r requirements.txt
```

If the setup does not work on your system, create a virtual environment and try again:

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Run

```bash
python convert_kanji.py
```

## Output

- joyo_kanji_readings.json
